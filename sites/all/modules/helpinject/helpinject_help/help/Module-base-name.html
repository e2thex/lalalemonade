The base name (for use in the PHP code) of the new module. This must only consist of lowercase alphabetical characters (a-z), and the underscore (_).

This new module will contain the help files that have been injected, packaged in a way that is usable by the Advanced Help module.

The newly generated module will also have a generated module, written in PHP, that must be enabled, just like any other module. This generated module is needed for the help icons to be injected.

Thus the module name that is chosen for this field must be original, and should only contain lowercase letters and underscores (_).

If you are using HelpInject to generate help files for a module you maintain, the suggestion is to follow the pattern yourmodulename_help.
